Parsing_beautifulsoup_requests
Этот проект демонстрирует примеры использования библиотек BeautifulSoup и requests для парсинга сайтов. В рамках проекта были проанализированы следующие сайты:
1. https://knigopoisk.org/
2. https://books.toscrape.com/

Структура проекта
1. Parsing_beautifulsoup_requests.ipynb - файл с примером использования BeautifulSoup и requests для парсинга сайта Книгопоиск.
2. Parsing_beautifulsoup_requests_books.ipynb - файл с примером использования BeautifulSoup и requests для парсинга сайта books.toscrape.com
3. requirements.txt - файл с зависимостями проекта

Использование
Каждый из Jupyter Notebook файлов содержит примеры использования BeautifulSoup и requests для парсинга сайтов. В результате выполнения кода создаются объекты pd.DataFrame, содержащие полученные данные.

Требования
Для запуска проекта необходимо установить зависимости, перечисленные в файле requirements.txt.
